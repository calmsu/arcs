Keydowns on the document element are bound here. We'll grab 37 & 39 and let the others pass through.

 
    keydownDelegate: (e) =>
        if e.keyCode == 37
            @.prevResource()
            e.preventDefault()
            return false
        if e.keyCode == 39
            @.nextResource()
            e.preventDefault()
            return false

Set the model by collection index.
